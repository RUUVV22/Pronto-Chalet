# نظام حجوزات الشاليه

تطبيق عربي خاص لإدارة حجوزات الشاليه باستخدام `Node.js` و`Express` و`HTML` و`CSS` و`JavaScript` الخام.

## المميزات

- واجهة عربية بالكامل مع اتجاه `RTL`
- تسجيل دخول خاص مع إنشاء أول حساب إداري عند التشغيل الأول
- إمكانية إنشاء حسابات إضافية وتغيير كلمة المرور من داخل النظام
- لوحة تحكم متجاوبة للهواتف والكمبيوتر
- تقويم للحجوزات، جدول للحجوزات، بحث بالاسم، وتعديل أو حذف للحجز
- REST API يعمل على ملف JSON محلي للحجوزات

## التشغيل

```bash
npm install
npm start
```

افتح الرابط الذي يظهر في الطرفية. سيستخدم `http://localhost:3000` عند توفره،
أو أقرب منفذ متاح إذا كان المنفذ `3000` مستخدما.

## ملفات البيانات

- `data/bookings.json` لتخزين الحجوزات
- `data/auth/users.json` لتخزين الحسابات الإدارية
- `data/auth/sessions.json` لتخزين الجلسات
